About this guide

Welcome to the DevProgress Style Guide. This guide is intended for anyone working on DevProgress projects. The purpose is to provide a set of building blocks to quickly create websites and applications.

The styles contained within are based on Bootstrap so any components found in Bootstrap can be used.

Getting started

To get started using the styles, simply download the css, fonts and js files from the dist directory from the style guide github repo and include them in your html files.
